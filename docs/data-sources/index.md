# Data Sources

This section contains documentation for all F5 Distributed Cloud Terraform data sources.

## Available Data Sources

Data sources allow you to fetch information about existing F5XC resources for use in your Terraform configurations.

## Usage

Each data source documentation page includes:

- **Description**: What information the data source retrieves
- **Example Usage**: Terraform configuration examples
- **Argument Reference**: Required and optional arguments
- **Attribute Reference**: Available attributes from the fetched data

## Common Use Cases

- Reference existing resources without managing them
- Look up resource IDs for use in other resources
- Query configuration details from F5XC
- Import existing infrastructure information into Terraform state
