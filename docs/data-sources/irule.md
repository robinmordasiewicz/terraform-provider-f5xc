# f5xc_irule (Data Source)

Manages iRule in a given namespace. If one already exists it will give an error. in F5 Distributed Cloud.

~> **Note** For more information about this data source, please refer to the [F5 XC API Documentation](https://docs.cloud.f5.com/docs/api/).

```terraform
# Irule Data Source Example
# Retrieves information about an existing Irule

# Look up an existing Irule by name

  namespace = "system"
```
