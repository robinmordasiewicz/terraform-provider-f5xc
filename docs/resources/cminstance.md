## Argument Reference

The following arguments are required:

`name` - (Required) Name of the Cminstance. Must be unique within the namespace (`String`).

`namespace` - (Required) Namespace where the Cminstance will be created (`String`).

The following arguments are optional:

`api_token` - (Optional) Secret. SecretType is used in an object to indicate a sensitive/confidential field. See [API Token](#api-token) below for details.

`ip` - (Optional) IPv4 Address. IPv4 Address in dot-decimal notation. See [IP](#ip) below for details.

`username` - (Optional) Username. Username for the Central Manager instance (`String`).
