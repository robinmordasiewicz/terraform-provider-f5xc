# f5xc_forward_proxy_policy (Resource)

Manages a ForwardProxyPolicy resource in F5 Distributed Cloud for forward proxy policy configuration.

~> **Note** For more information about this resource, please refer to the [F5 XC API Documentation](https://docs.cloud.f5.com/docs/api/).

## Example Usage

```terraform
# Forward Proxy Policy Resource Example
# Manages a ForwardProxyPolicy resource in F5 Distributed Cloud for forward proxy policy configuration.

# Basic Forward Proxy Policy configuration
resource "f5xc_forward_proxy_policy" "example" {
  name      = "example-forward-proxy-policy"
  namespace = "system"

  labels = {
    environment = "production"
    managed_by  = "terraform"
  }

  annotations = {
    "owner" = "platform-team"
  }

  # Forward Proxy Policy configuration
  proxy_label_selector {
    expressions = ["app in (web, api)"]
  }

  drp_http_connect {
    any_proxy {}
    rule_list {
      rules {
        metadata {
          name = "allow-external"
        }
        spec {
          action = "ALLOW"
          dst_list {
            any_dst {}
          }
        }
      }
    }
  }
}
```
