# Resources

This section contains documentation for all F5 Distributed Cloud Terraform resources.

## Available Resources

The F5XC provider includes resources for managing:

- **Application Security**: WAF, API protection, bot defense
- **Load Balancing**: HTTP/HTTPS load balancers, origin pools, health checks
- **Networking**: Virtual networks, sites, network policies
- **DNS**: DNS zones, records, load balancing
- **Infrastructure**: Cloud sites (AWS, Azure, GCP), edge sites
- **Identity & Access**: Service policies, API credentials, namespaces

Browse the resources in the sidebar to find detailed documentation for each resource type.

## Usage

Each resource documentation page includes:

- **Description**: What the resource manages
- **Example Usage**: Terraform configuration examples
- **Argument Reference**: Required and optional arguments
- **Attribute Reference**: Exported attributes
- **Import**: How to import existing resources
