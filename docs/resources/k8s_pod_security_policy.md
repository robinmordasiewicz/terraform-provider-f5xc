## Argument Reference

The following arguments are required:

`name` - (Required) Name of the K8SPodSecurityPolicy. Must be unique within the namespace (`String`).

`namespace` - (Required) Namespace where the K8SPodSecurityPolicy will be created (`String`).

The following arguments are optional:

`annotations` - (Optional) Annotations to apply to this resource (`Map`).

`labels` - (Optional) Labels to apply to this resource (`Map`).

> **Note:** One of the arguments from this list "psp_spec, yaml" must be set.

`psp_spec` - (Optional) Pod Security Policy Specification. Form based pod security specification. See [Psp Spec](#psp-spec) below for details.

`timeouts` - (Optional) See [Timeouts](#timeouts) below for details.

`yaml` - (Optional) K8s YAML. K8s YAML for Pod Security Policy (`String`).
